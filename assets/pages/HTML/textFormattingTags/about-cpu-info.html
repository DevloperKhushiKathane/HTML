What is Primary Memory?

Primary memory, also known as main memory or RAM, is a computer's internal memory that the CPU can directly access. It's used to store data and instructions that the computer is actively using, enabling quick access and processing.

What is Secondary Memory?

Secondary memory, also known as auxiliary or external memory, is a type of non-volatile computer memory that is not directly accessed by the CPU. It provides long-term data storage for files, applications, and the operating system. It is the memory thet stores the instruction that the CPU executes. It is also known as the "program memory".It is also known as secondary memory or RAM(Random Access Memory).

What is Processor?

A processor also known as CPU(Central Processing Memory).It is the brain of the computer.It's the hardware component that executes instruction about software and perform calculations,esentially controling all the operation.
